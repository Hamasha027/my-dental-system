# 📊 دليل إنشاء البيانات الاختبارية

## طريقة سريعة وسهلة

### ✅ الطريقة الأولى: عبر واجهة المستخدم (الأسهل)

1. **افتح التطبيق وتأكد من أنك في لوحة التحكم**
2. **اذهب إلى الرابط:** `http://localhost:3000/dashboard/generate-test-data`
3. **انقر الزر الأزرق:** "إضافة البيانات الوهمية"
4. **انتظر الرسالة:** ✅ "تمت إضافة البيانات الوهمية بنجاح!"

### 🎯 ماذا سيتم إضافته؟

✅ **5 موظفين وهميين:**
- محمد علي (پەرستار) - 800,000 دينار
- فاتيمة مراد (دندان پزیشک) - 2,000,000 دينار
- حسن كريم (تێکنیشین) - 600,000 دينار
- نور محمد (پەرستار) - 750,000 دينار
- سارا حسن (سەکرتێر) - 500,000 دينار

✅ **6 شهور من البيانات:**
- آخر 6 شهور من الفترة الحالية
- رواتب ومقدمات لكل موظف
- سجلات تاريخ الرواتب

---

## 🧪 كيفية الاختبار

### اختبار 1: عرض الموظفين
```
1. اذهب: داشبورد → كارمندەكان
2. ستجد 5 موظفين في الجدول
3. يمكنك تحديث وحذف الموظفين
```

### اختبار 2: عرض ڕاپۆرتی موچە (تقرير الرواتب)
```
1. اذهب: كارمندەكان → ڕاپۆرتی موچە
2. نافذة القائمة المنسدلة ستظهر الشهور السابقة:
   - الشهر الحالي
   - الشهر الماضي
   - وهكذا... لمدة 6 شهور
3. انقر على شهر مختلف لترى البيانات
```

### اختبار 3: اختبار زر "كوتاي ماني" (إنهاء الشهر)
```
1. اذهب: كارمندەكان
2. انقر "كوتاي ماني"
3. سيظهر تحذير يطلب التأكيد
4. انقر "بەڵێ، مانگ كوتاي بهێنە"
5. ستُحفظ البيانات ورسوم الرواتب = 0
```

### اختبار 4: عرض البيانات المحفوظة
```
1. اذهب: كارمندەكان → ڕاپۆرتی موچە
2. اختر "ڕاپۆرتی موچەکانی گشتی" في الأعلى
3. ستجد الشهر الذي كملته في الجدول
4. انقر "پرێنت" لتحميل PDF
```

## 🔧 إذا أردت حذف البيانات الوهمية

### حذف موظف واحد:
```
1. اذهب: كارمندەكان
2. انقر الزر الأحمر (سلة المهملات) بجانب الموظف
3. أكد الحذف
```

### حذف جميع البيانات (خطر!):
```
⚠️ يجب حذف قاعدة البيانات والبدء من جديد
أو استخدم أداة database management للحذف المباشر
```

## ⚡ نصائح للاختبار الفعال

1. **لا تنسَ:** انقر "كوتاي ماني" لحفظ الشهر قبل الانتقال للشهر التالي
2. **للتحقق:** اذهب لڕاپۆرتی موچە واختبر المنسدلة
3. **للطباعة:** انقر البحث والطلب وستظهر خيارات الطباعة
4. **للتصحيح:** إذا أردت بيانات مختلفة، احذف الموظفين وأضفهم يدويًا

## 🐛 إذا حدثت مشكلة

### المشكلة: لا تظهر البيانات
- ✅ تأكد من تحديث الصفحة (F5)
- ✅ تأكد من اتصالك بقاعدة البيانات
- ✅ تحقق من وحدة طلب الخطأ في أسفل الصفحة

### المشكلة: الأزرار لا تعمل
- ✅ حاول إعادة تحميل الصفحة
- ✅ امسح ذاكرة التخزين المؤقتة (Ctrl + Shift + Delete)
- ✅ أعد تشغيل الخادم

### المشكلة: البيانات تكررت
- ✅ هذا طبيعي، النظام يتجاهل التكرارات
- ✅ لا تقلق، يمكنك الاختبار بهدوء
